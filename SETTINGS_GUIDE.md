## üí° **TIPS FOR BEST RESULTS**

### **For Logo:**
- Use PNG with transparent background
- Recommended size: 200x60px to 400x120px
- Keep it simple and readable

### **For Favicon:**
- Use square image (32x32px or 64x64px)
- Simple design works best (too much detail won't show)
- Use ICO or PNG format

### **For Colors:**
- Test on both light and dark backgrounds
- Ensure good contrast for readability
- Use a color picker tool for hex codes

### **For SEO:**
- Keep titles under 60 characters
- Keep descriptions between 150-160 characters
- Use relevant keywords naturally

### **For Contact Info:**
- Use full international format for phone: +91 XXXXX XXXXX
- Include complete address with postal code
- Use professional email address

## ‚ùì **TROUBLESHOOTING**

**Q: I changed a setting but don't see it on the website**
- A: Make sure you clicked "Save Changes"
- A: Refresh the page (F5 or Cmd+R)
- A: For favicon, do a hard refresh (Cmd+Shift+R)

**Q: My logo looks stretched or distorted**
- A: Upload a logo with the correct aspect ratio (wide, not square)
- A: Recommended: 3:1 or 4:1 ratio (width:height)

**Q: Colors aren't changing**
- A: Make sure you're using hex color codes (#XXXXXX)
- A: Click "Save Changes" after entering the color
- A: Clear browser cache if needed

**Q: Social media icons not showing**
- A: Make sure you entered the full URL (https://instagram.com/username)
- A: Click "Save Changes"
- A: Refresh the page
